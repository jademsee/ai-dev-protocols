# /prescribe

Prescribe — recommend treatments, no surgery.

## Steps

1. Diagnose the current file and produce a prioritized improvement plan
2. Do NOT write any code
3. List every improvement candidate in priority order with:
   - Category (correctness / concurrency / performance / clarity)
   - Estimated impact (high / medium / low)
   - Files that would be touched
   - Any Hard Stop risks
4. Stop after the plan. Wait for me to choose a loop mode
