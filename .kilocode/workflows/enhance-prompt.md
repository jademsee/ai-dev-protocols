# /enhance-prompt

Enhance and improve the user's prompt.

## Steps

1. Read the user's request carefully
2. Generate an enhanced, more comprehensive version that is:
   - More specific and actionable
   - Technically precise
   - Complete with edge cases and constraints
   - Clear about expected outcomes
3. Output ONLY the enhanced prompt with no preamble, explanations, or meta-commentary
